数值运算

基本没什么变化 加减乘除模 + - * / %

赋值运算

  • 如x += 10等于 x = x + 10
  • 还有如%= /=等
  • x++和++x是不同的，前者是先运算再自增，反之不同
    • 如x = 5;y = x++;//x=6, y=5
    • 而x = 5;y = ++x;//x=6, y=6

int, float, double

int a = 42;

前面可以用modifiers修饰:

  • signed：可用于正负数
  • unsigned：只可用于正数
  • short：默认大小的一半
  • long：默认大小的两倍

一共有三种float：

  • float：4B
  • double：8B
  • long double：8B或者16B

  • Address-of: & 返回内存地址
  • Contents-of: * 返回此地址上的值

  • Overloading：允许创建相同名字但是不同参数的函数

传递参数

有两种传递方式，Value和Reference，

  • Value：传递的是变量的copy，这意味着原始参数不会被更改
  • Reference：传递的是地址，会被更改

类和对象

面向对象编程（Object Oriented Programming,OOP）是一种试图使编程更靠近真实世界思考的编程方式。

对象

对象包含，

  • 本体（identity）： 对象的身份标识（唯一）
  • 属性（attribute）：描述了对象的特征
  • 行为（behavior）：对象的动作行为

类

对象是由类创建的实例（instance），也就是类是对象的蓝图。类包含方法（method），是属于类的函数。

  • public
  • Abstraction（抽象化）
  • 数据隐藏
  • 构造器

构造器是在创建对象的时候运行的函数，所以在初始化的时候传递参数很有用。
