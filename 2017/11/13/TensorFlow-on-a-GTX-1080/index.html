TensorFlow on a GTX 1080


Ubuntu 16.03 安装 CUDA、NVIDIA驱动，CUDNN及GPU版TensorFlow。

GPU 支持的TensorFlow让算力大幅提升，但是安装好一切支持却不那么容易！其实主要是三个东西：

  1. Nvidia 驱动：显卡驱动
  2. CUDA Toolkit CUDA工具箱
  3. CUDNN：CUDA Deep Neural Network library 神经网络库函数
