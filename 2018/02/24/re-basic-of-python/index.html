发表于 2018-02-24 | 阅读次数

本文主要关于python的正则表达式的符号与方法。

  • findall: 找寻所有匹配，返回所有组合的列表
  • search: 找寻第一个匹配并返回
  • sub: 替换符合规律的内容，并返回替换后的内容

.：匹配除了换行符以外的任意字符

*：匹配前一个字符0次或者无限次

?：匹配前一个字符0次或者1次

.*：贪心算法

.*?：非贪心算法

()：括号内结果返回

re.S使得.的作用域包括换行符”\n”
