如下设置默认都在每一台机器上都要进行。

配置hadoop用户

每台机器之间以及每台机器与自己的localhost都需配置ssh免密码登录。
如未安装ssh需 sudo apt install openssh-server

再用ssh-copy-id来设置免密码登录，如
